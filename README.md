# German-AI-Skills

A small collection of Agent Skills for working with German (and mixed German/English) text.

Currently includes:

- **`de`** — detects the input language, corrects grammar/spelling/wording, and always returns both a corrected German and a corrected English version.


# Install skills

```
npx skills add https://github.com/ayonious/German-AI-Skills/tree/main/skills/de
```
